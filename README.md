# 🌱 Crop Recommendation System Using Machine Learning

A web-based application that recommends the most suitable crop 
to grow based on soil nutrients and climate conditions.

## Features
- Predicts best crop based on N, P, K, Temperature, 
  Humidity, pH, and Rainfall
- Built with Flask web framework
- Machine Learning model trained on Kaggle crop dataset
- MinMax and Standard Scaler for data preprocessing

## Tech Stack
- Python, Flask
- Scikit-learn (Random Forest)
- Pandas, NumPy
- HTML, CSS

## How to Run
pip install -r requirements.txt
python app.py

## Dataset
Kaggle - Crop Recommendation Dataset
